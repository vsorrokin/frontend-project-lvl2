## Installation:

Run commands:

    npm install
    npm link

## Usage:

General information:

    Usage: gendiff [options] <filepath1> <filepath2>

    Compares two configuration files and shows a difference.

    Options:
        -V, --version        output the version number
        -f, --format [type]  output format
        -h, --help           display help for command
